\subsection{Detailed Description}
An input stream decompressing data compressed with the bz2 algorithm. 

The actual decompression is achieved in the underlying \hyperlink{classbz2inbuf}{bz2inbuf} stream buffer; you should read its documentation before using this class.

The data is read from a supplied stream buffer.
