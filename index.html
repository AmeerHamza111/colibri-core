Colibri Core is a set of tools as well as a C++ and Python library for working with basic linguistic constructions such as n-grams and skipgrams (i.e patterns with one or more gaps, either of fixed or dynamic size) in a quick and memory-efficient way. At the core is the tool colibri-patternmodeller which allows you to build, view, manipulate and query pattern models.

The Colibri software is developed in the scope of the Ph.D. research project Constructions as Linguistic Bridges. This research examines the identification and extraction of aligned constructions or patterns across natural languages, and the usage of such constructions in Machine Translation. The aligned constructions are not identified on the basis of an extensive and explicitly defined grammar or expert database of linguistic knowledge, but rather are implicitly distilled from large amounts of example data. Our notion of constructions is broad and transcends the idea of words or variable-length phrases.

The software consists out of various tools and an API for C++ as well as Python, the tools and Python API are documented in the documentation, the C++ API reference is documented separately. Depending on the size of your input corpus, certain tasks may take considerable memory. We recommend a Linux machine with at least 8GB RAM.

Colibri Core has the following facilities:

  • Pattern modelling - Build pattern models from corpus data. Models may contain n-grams and/or skipgrams and may be either unindexed or indexed (maintaining precise references to the covered tokens).
  • Model visualisation - Visualisation of models in various ways, including token coverage reports, histograms, and graphs that show the relationships between patterns
  • Model querying - Interactively query generated models for specific patterns and relations between patterns
  • Model/corpus comparison - Match patterns generated from one corpus to another corpus.

Most of the code is written in C++, offering a clean API interface. A Python wrapper binding with the C++ library is available, giving access to fast and memory-efficient pattern models from a high-level language.
